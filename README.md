Sales Analysis Dashboard

Project Overview

This project is a Power BI dashboard that analyzes sales performance, profit, and quantity across different regions, products, and salespersons.

 Features

* Total Sales, Profit, and Quantity KPIs
* Sales by Region and Category
* Sales vs Profit Analysis
* Sales Trend Over Time
* Interactive Filters (Date, Region, Category)

 Tools Used

* Power BI
* Data Visualization
